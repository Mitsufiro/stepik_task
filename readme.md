# Анализ логов веб-сервера с использованием PySpark

## Описание проекта

Данный проект выполняет анализ логов веб-сервера с помощью Apache Spark и PySpark. Код предоставляет базовую аналитику HTTP-запросов, включая статистику по IP-адресам, методам запросов, кодам ответов и объемам передаваемых данных.

## Структура данных

Анализируемый CSV-файл содержит следующие поля:
- ip - IP-адрес клиента
- timestamp - временная метка запроса
- method - HTTP-метод (GET, POST, PUT, DELETE)
- url - запрашиваемый URL-адрес
- response_code - HTTP-код ответа сервера
- response_size - размер ответа в байтах
